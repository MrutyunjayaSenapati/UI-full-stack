Inter-document Link

  • Navigate to file

You can create a link that navigates the user to any specific file in the project or a remote location. You can link to any type of file on the page [html, pdf, doc, ppt, xls, jpg, etc.].
Every file can't open in the browser. It requires the relevant plugins to be installed. Syntax: <a href="./docs/tutorial.pdf">Tutorial</a>

  • Image
  • Document
  • Navigate to URL
You can configure a link that navigates the user to a remote URL.
URL format requires an exact match of protocol and domain name.
  • Open specific application
You can configure a link that triggers a specified app on click.
- Applications that are supported by the browser can be invoked from the browser.
- Browser applications include:
    a) phone            tel:number
    b) email            mailto:email_address
    c) skype            skype:profile_name?action
    d) WhatsApp         https://wa.me/number

  • Trigger a JavaScript function

HyperLink can point towards a JavaScript function.

It can be user-defined or a built-in function.
